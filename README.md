# About

Emmett is an ETL tool that supports drop-in Logstash functionality through a config setting.

# Usage

```

(=^•ω•^=)

Emmett 0.1.0
Andrew Banchich

USAGE:
    emmett [FLAGS]

FLAGS:
    -d, --debug
    -h, --help       Prints help information
    -V, --version    Prints version information

```
